# Parallel Life AI - 使用指南

## 🎯 如何使用

### 第一步: 配置API密钥

本项目通过 [OpenRouter](https://openrouter.ai/) 调用大模型，**默认使用免费模型，无需充值**。

1. 访问 https://openrouter.ai/ 注册/登录
2. 在 https://openrouter.ai/keys 创建 API Key（以 `sk-or-v1-` 开头）
3. 复制 `.env.local.example` 为 `.env.local`
4. 将 API Key 填入 `OPENROUTER_API_KEY` 字段
5. `OPENROUTER_MODEL` 默认已是免费模型 `nvidia/nemotron-3-super-120b-a12b:free`（id 以 `:free` 结尾即免费）。想用更强的付费模型时改成对应 slug 并到 OpenRouter 充值额度即可。

> 进阶：设置 `OPENROUTER_BASE_URL` 可指向任意 OpenAI 兼容网关。例如本机安装 [Ollama](https://ollama.com) 后设为 `http://localhost:11434/v1`，即可完全离线、零成本运行。

### 第二步: 启动项目

```bash
npm run dev
```

访问 http://localhost:3000

### 第三步: 生成平行人生

1. **点击"开始探索"** 进入表单页面
2. **填写个人信息**:
   - 年龄 (16-60岁)
   - 性别
   - 国家/地区
   - 当前职业
   - 当前生活状态
   - 关键决定 (至少20个字符)

3. **点击"开始生成平行世界"**
4. **等待AI生成** (约10-30秒)
5. **查看结果**:
   - 时间线视图: 对比两条路线的各个阶段
   - 故事模式: 阅读文学化的人生叙事
   - 分支树: 可视化的人生分叉图

6. **分享你的平行人生**:
   - 点击右上角"分享"按钮
   - 复制链接或分享到社交平台

## 💡 使用技巧

### 好的关键决定示例

✅ **具体明确**:
- "如果当初选择去日本留学而不是在国内读研"
- "如果辞职创业继续做互联网产品"
- "如果接受北京的offer去上海发展"

❌ **过于模糊**:
- "如果换个选择"
- "如果不一样"

### 获得更好的结果

1. **详细描述当前状态**: 让AI更了解你的背景
2. **明确关键决定**: 越具体,生成的时间线越真实
3. **尝试不同角度**: 同一个决定可以多次生成,每次结果可能不同

## 🔧 常见问题

### Q: API调用失败怎么办?

A: 检查以下几点:
1. 确认 `.env.local` 文件存在且 `OPENROUTER_API_KEY` 配置正确
2. 免费模型偶尔会排队或返回异常，程序已内置自动重试，稍等重试即可
3. 若提示"额度不足(402)"，把 `OPENROUTER_MODEL` 换成以 `:free` 结尾的免费模型
4. 重启开发服务器、检查网络连接

### Q: 生成的内容不理想?

A: 可以尝试:
1. 重新生成一次(温度参数有随机性)
2. 优化输入信息,提供更多细节
3. 调整关键决定的描述方式

### Q: 如何清除历史记录?

A: 
- 在首页"最近生成"区域,点击卡片上的删除按钮
- 或清除浏览器localStorage

### Q: 可以保存多少次记录?

A: 本地最多保存10条记录,超出后会自动删除最早的记录。

## 🎨 自定义

### 修改主题色

编辑 `app/globals.css`:

```css
:root {
  --primary: #6366f1;      /* 主色调 */
  --secondary: #a855f7;    /* 次要色调 */
  --accent: #06b6d4;       /* 强调色 */
}
```

### 调整AI生成风格

编辑 `prompts/templates.ts`,修改Prompt模板中的指令。

## 📱 移动端适配

项目已完全响应式设计,支持:
- 📱 手机 (< 768px)
- 📱 平板 (768-1023px)
- 💻 桌面 (≥ 1024px)

## 🚀 性能优化建议

1. **图片懒加载**: 后续可添加图片生成功能时实现
2. **API缓存**: 对相同输入可以缓存结果
3. **代码分割**: 已使用Next.js动态导入
4. **CDN加速**: 部署时使用Vercel Edge Network

## 🔐 隐私说明

- 所有数据存储在浏览器localStorage
- 不会上传到服务器(除API调用外)
- 清除浏览器数据即可完全删除记录
- 建议使用无痕模式保护隐私
